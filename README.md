# IDEA shortcuts

Some JetBrains IDEA useful shortcuts:

- `Ctrl + Q` - see documentation
- `Ctrl + J` - add template
- `Ctrl + Alt + L` - reformat
- `Ctrl + Shift + Down/Up` - move statement down/up
- `Ctrl + Alt + S` - settings
- `Ctrl + B` - jump to declaration/usage of the class/method
